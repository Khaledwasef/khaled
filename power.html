Integrated Power Supply with Home Security System

This innovative project combines a versatile power supply unit with a comprehensive home security system. The power supply provides both variable voltage (0-32V) and fixed voltage outputs (±5V, ±9V, ±12V) with a maximum current of 3A. The integrated security system features fire and gas detection capabilities, addressing critical safety concerns in residential environments.

Power Supply Implementation

The power supply implementation involves connecting the 220V AC source to a transformer through a switch. The transformer output is connected to a bridge rectifier to convert AC to DC. A 4700µF capacitor smooths the signal, which is then regulated by buck converters (for variable output) and voltage regulators (for fixed outputs). The digital panel counter displays the output values.

Security System Implementation

The home security system features two critical detection systems:

In 2022, Egypt experienced 49,341 fire incidents, with 34.5% occurring in residential settings. This alarming statistic underscores the critical need for reliable early detection systems to protect homes and families from fire hazards.

Fire Detection System

The fire detection system uses an IR sensor to detect fire and triggers a buzzer alarm. Components include:

  • IR sensor for flame detection
  • 5V power source
  • Buzzer for audible alarm
  • Transistor for circuit control
  • Resistors and wiring

Gas Detection System

The gas detection system uses an MQ5 sensor to detect dangerous gas leaks, which can lead to explosive situations. The system includes:

  • MQ5 gas sensor module
  • 5V buzzer for alarm
  • NPN transistor (BC547)
  • 5V power source
  • Wiring and resistors

Conclusion

This integrated system provides both a versatile power solution for electronics projects and critical safety features for home environments. The power supply offers both variable and fixed outputs suitable for various applications, while the security system addresses the alarming rate of residential fires in Egypt, particularly those caused by gas leaks that can lead to catastrophic explosions.

The project demonstrates effective team collaboration across multiple technical domains, from power electronics to sensor-based safety systems. By combining these elements into a single solution, the project delivers practical value for both electronics enthusiasts and homeowners concerned with safety.
